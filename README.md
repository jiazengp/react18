# React 18

## Description

React 18 experience and learning.

## Development Setup

Clone the repo, and install dependencies:

``` sh
yarn
# npm install
# pnpm install
```

Open another terminal, and start developing:

``` sh
yarn dev
# npm run dev
# pnpm dev
```
